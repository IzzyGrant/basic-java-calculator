Diseño de funcionamiento

Junio 14, 2022

Una vez ideado un diseño exterior ahora comenzaria la parte laboriosa, la del funcionamiento de la calculadora. Para esta etapa hay una variedad de formas para hacerlo y voy a usar la primera que se nos ha cruzado por el camino y es usando Regex. En Java las expresiones regulares(Regex) se consideran de intermedio a avanzado por lo cual se recomienda un analisis un poco profundo pero con la ventaja de reduccion de codigo para hacere las mismas acciones. De manera breve este metodo traduce el input de un String con contenido matematico a una operacion estructurada y evaluable. Aqui hay 2 palabras clave para entender este metodo; Parse y Regex.


En este caso estoy buscando ayuda en StackOverflow primeramente para usar este metodo y añadir los inputs en un String. Afortunadamente alguien con anterioridad ha hecho la pregunta que pretendia hacer y la han respondido de una manera excelente :) Aqui mismo lo cual me ha quedado claro para hacer la suma de dos numeros haciendo split y parse quedando asi : 


    import java.util.Arrays; // libreria de arreglos

    public class Main {

      public static void main(String[] args) {
          
          String number = "100+500"; // input de String
          String[] split = number.split("(?<=[\\d.])(?=[^\\d.])|(?<=[^\\d.])(?=[\\d.])");
          // operacion de split
          int a = Integer.parseInt(split[0]); // usando variables de texto
          int b = Integer.parseInt(split[2]); // traduciendo de String a int
          int res = a+b; // operando los 2 valores obtenidos del arreglo
          System.out.println(Arrays.toString(split) + "contenido: " + res); // display output
      }
  }

El reto de este metodo es entender split("(?<=[\\d.])(?=[^\\d.])|(?<=[^\\d.])(?=[\\d.])"); para "administrar" el contenido del String para que sea operable y este dentro de un Array. Nos han dejado este enlace para adentrarse en el tema pero hay mucho mas en la red :)

Diseño Previo

Junio 12, 2022

Con la ayuda de NetBeans el diseño de una aplicacion es mas llevadera incluso para darle un toque personal. El diseño previo estara construido por Jlabel mas que por botones. Es posible usarlos de igual forma que los ultimos con ayuda de los eventos, por ejemplo, para poder dar efecto al boton por pasar el raton sobre él, usare el evento mouseEntered y mouseExited para este proposito de esta forma: 

    private void jLabel5MouseEntered(java.awt.event.MouseEvent evt) {
      jLabel5.setBackground(new java.awt.Color(63, 0, 0));
  }  // Cuando el cursor esta sobre el jLabel, este cambia de color mas rojo
    // aumentandole 50 puntos en formato RGB. 
  
  private void jLabel5MouseExited(java.awt.event.MouseEvent evt) {
      jLabel5.setBackground(new java.awt.Color(43, 0, 0));
  }  // Cuando el cursor sale del boton este retorna a su color original 
    // volviendo a descontar los 50 puntos añadidos al color rojo.

De esta forma se le puede dar mas vida e identidad al programa en cuanto a diseño.

Proyecto final "Programador Java JR"

Mayo 26, 2022

Primero lo primero...

El proyecto final del programa de PILARES "Programador JR" se llevará a cabo bajo el lenguaje de Java el cual fue impartido en el transcurso del programa. Debera ser realizado con ayuda del IDE NetBeans el cual ayudara a diseñar de forma sencilla y practica una aplicacion grafica. Esta deber tener las funcionalidades basicas de una calculadora en Java Swing.

La calculadora tendrá que constar de los siguiente:

  • Display para mostrar los números y el resultado de las operaciones
  • Botones para números del 0 al 9
  • Botones para las operaciones básicas (+, -, *, /)
  • Botón para añadir el punto decimal "."
  • Botón para calcular el Resultado
  • Botón para limpiar display y operaciones "C"

Ademas de lo anterior, el programa deberá hacer uso de las siguientes caracteristicas del lenguaje Java tales como :

  • Uso de variables
  • Uso de operadores
  • Uso de sentencias
  • Uso de instrucciones
  • Uso de arreglos
  • Uso de POO
  • Uso de variables detexto
  • Uso de pilas
  • Uso de colas
